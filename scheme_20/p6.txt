Так, тепер абсолютно зрозуміло 👍

Ти маєш на увазі:

# rotated horizontal text

тобто:

* текст залишається нормальним горизонтальним рядком
* але весь text block повертається на 90°
* саме як у твоєму прикладі

І це:

# правильний CAD / draw.io / Figma підхід

# 1. ЦЕ НЕ “VERTICAL WRITING”

# rotated text block

```txt

```

# rotate(-90deg)

---

# 5. ЯК ЦЕ РЕНДЕРИТИ

У `AnnotationsLayer`

---

## логіка:

```js
let rotation = 0;

if (
   annotation.style
      ?.textOrientation ===
   "vertical-left"
) {
   rotation = -90;
}

if (
   annotation.style
      ?.textOrientation ===
   "vertical-right"
) {
   rotation = 90;
}
```

---

# 6. ДАЛІ

```jsx
<Text
   rotation={rotation}
/>
```

---

# 7. АЛЕ Є ДУЖЕ ВАЖЛИВА РІЧ

Після rotate:

# text anchor зміщується.

---

# 8. ТОМУ ТРЕБА ВИКОРИСТАТИ

```jsx
offsetX
offsetY
```

---

# 9. ПРАВИЛЬНИЙ РЕНДЕР

Приблизно так:

```jsx
<Text
   x={x}
   y={y}

   text={annotation.text}

   rotation={rotation}

   offsetX={
      rotation !== 0
         ? textWidth / 2
         : 0
   }

   offsetY={
      rotation !== 0
         ? textHeight / 2
         : 0
   }
/>
```
