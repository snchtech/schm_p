Тепер у тебе стало навіть **правильніше архітектурно** 👍
SVG-іконки в панелі — це **тільки UI**, а **Konva Stage — це canvas-редактор**. Їх не треба змішувати.

Тобі потрібно реалізувати **один простий потік даних**:

```
ToolPalette (SVG icon click)
        ↓
activeTool = "twoRectangle"
        ↓
DiagramEditor
        ↓
click on Stage
        ↓
create element
        ↓
S.symbols.push(...)
        ↓
DiagramSymbols render
```

Я покажу **мінімальну і чисту реалізацію**.

# 1️⃣ Registry (типи елементів)

registry має тільки **дані**, не компоненти.

```javascript
export const ELEMENT_REGISTRY = {
  twoRectangle: {
    title: "Two rectangles",
    defaults: {
      w: 120,
      h: 60,
    }
  },

  circle: {
    title: "Circle",
    defaults: {
      r: 40
    }
  }
}
```

# 5️⃣ Додаємо element на Stage

```javascript
import { ELEMENT_REGISTRY } from "../graph/registry"

const handleStageMouseDown = (e) => {

  if (!activeTool) return

  // щоб не створювало при кліку по element
  if (e.target !== e.target.getStage()) return

  const stage = e.target.getStage()
  const pos = stage.getPointerPosition()

  const tool = ELEMENT_REGISTRY[activeTool]

  const newElement = {
    id: crypto.randomUUID(),
    type: activeTool,
    x: pos.x,
    y: pos.y,
    ...tool.defaults
  }

  S.setSymbols(prev => [...prev, newElement])
}
```

# 6️⃣ Підключаємо до Stage

# ⭐ Дуже важлива порада

Зроби **одну функцію createElementFromTool**

щоб код був чистим.

```
graph/createElement.js
```

```javascript
export function createElementFromTool(type, x, y) {

  const tool = ELEMENT_REGISTRY[type]

  return {
    id: crypto.randomUUID(),
    type,
    x,
    y,
    ...tool.defaults
  }
}
```

і в editor

```javascript
const el = createElementFromTool(activeTool, pos.x, pos.y)

S.setSymbols(prev => [...prev, el])
```
