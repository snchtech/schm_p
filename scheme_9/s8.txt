Добре 👍 Тепер зробимо **другий важливий крок — виправимо `DiagramSymbols`**, бо в твоєму поточному коді є серйозна React-помилка:

```js
symbols.map((el) => {

  const Shape = ELEMENT_COMPONENTS[el.type]

  useEffect(() => { ... })
```

❗ **React hooks не можна викликати всередині `map` або умов.**

Це може викликати:

* неправильний порядок hooks
* падіння компонентів
* дивні баги при додаванні/видаленні елементів

Тому правильна архітектура — **винести один символ в окремий компонент**.

```jsx
export const DiagramSymbols = (props) => {
```

# 4️⃣ Код DiagramSymbolItem

```jsx
"use client";

import React, { useEffect, useRef } from "react";
import { Group, Rect, Circle } from "react-konva";
import GraphLibrary from "../graph";

const ELEMENT_COMPONENTS = {
  rect: GraphLibrary.rect,
  circle: GraphLibrary.circle,
  line: GraphLibrary.line,
  text: GraphLibrary.text,
  twoRectangle: GraphLibrary.twoRectangle,
};

export default function DiagramSymbolItem({
  el,
  isAddingConnector,
  hoveredElement,
  hoveredAnchor,
  getAnchorPoints,
  setHoveredElement,
  setHoveredAnchor,
  handleDragMove,
  handleAnchorClick,
  onElementContextMenu,
  onMouseDown,
  isSelected,
  registerNodeRef,
}) {

  const groupRef = useRef(null);
  const Shape = ELEMENT_COMPONENTS[el.type];

  useEffect(() => {
    if (!registerNodeRef) return;

    registerNodeRef(el.id, groupRef.current);

    return () => {
      registerNodeRef(el.id, null);
    };
  }, [el.id, registerNodeRef]);

  return (
    <Group
      ref={groupRef}
      x={el.x}
      y={el.y}
      draggable
      onDragMove={(e) => handleDragMove(el.id, e)}
      onMouseEnter={() => isAddingConnector && setHoveredElement(el.id)}
      onMouseLeave={() => isAddingConnector && setHoveredElement(null)}
      onContextMenu={(e) =>
        onElementContextMenu(e, { type: "node", id: el.id })
      }
      onMouseDown={() => onMouseDown(el)}
    >
      {Shape && <Shape el={el} />}

      {/* selection outline */}
      {isSelected?.id === el.id && el.w && el.h && (
        <Rect
          x={-6}
          y={-6}
          width={el.w + 12}
          height={el.h + 12}
          cornerRadius={12}
          stroke="#3b82f6"
          strokeWidth={1}
          dash={[6, 4]}
          listening={false}
        />
      )}

      {/* anchors */}
      {hoveredElement === el.id &&
        getAnchorPoints(el).map((point, i) => (
          <Circle
            key={i}
            x={point.x - el.x}
            y={point.y - el.y}
            radius={6}
            fill={
              hoveredAnchor &&
              hoveredAnchor.x === point.x &&
              hoveredAnchor.y === point.y
                ? "yellow"
                : "green"
            }
            stroke="black"
            strokeWidth={1}
            onMouseEnter={() => setHoveredAnchor(point)}
            onMouseLeave={() => setHoveredAnchor(null)}
            onClick={() =>
              handleAnchorClick({
                ...point,
                parentId: el.id,
                kind: "symbolAnchor",
              })
            }
          />
        ))}
    </Group>
  );
}
```

# 5️⃣ Що ми виправили

Було:

❌

```js
symbols.map(() => {
   useEffect()
})
```

```js
symbols.map(() => <DiagramSymbolItem/>)
```
