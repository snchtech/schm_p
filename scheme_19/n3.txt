
# 1. Helper

```js
function getTerminalOrientation(attached, fallback) {

   if (attached?.anchor === 'top') return 'vertical'
   if (attached?.anchor === 'bottom') return 'vertical'

   if (attached?.anchor === 'left') return 'horizontal'
   if (attached?.anchor === 'right') return 'horizontal'

   return fallback || null
}
```

---

# 2. Під час detach

НЕ втрачати orientation.
