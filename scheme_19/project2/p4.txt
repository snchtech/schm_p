# Autonomous transaction

Для EVENT_QUEUE trigger.

---

# Чому

Щоб:

```text
remote notify
```

не впливав на головний INSERT.

---

# Приклад

```sql
CREATE OR REPLACE TRIGGER trg_event_queue
AFTER INSERT ON EVENT_QUEUE
FOR EACH ROW
DECLARE
    PRAGMA AUTONOMOUS_TRANSACTION;
BEGIN

    event_pkg.notify_change@DB21(
        :NEW.TABLE_NAME,
        :NEW.RECORD_ID,
        :NEW.EVENT_TYPE
    );

    COMMIT;

EXCEPTION
    WHEN OTHERS THEN
        NULL;
END;
/
```

---

# Оце вже дуже хороша схема.
