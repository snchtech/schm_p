Ок. Нижче **малий patch** для `handleConnectorContextMenu`, щоб по кліку на polyline знаходити **`segmentId`**, а не `segmentIndex`.

Це останній потрібний шматок для твоєї схеми з `segmentsMeta` і stable `segmentId`. 

---

## 1. helper: знайти найближчий сегмент до кліку

### `canvasController.js` або окремий helper

```js id="ctx-seg-01"
function distToSegment(px, py, x1, y1, x2, y2) {
  const dx = x2 - x1;
  const dy = y2 - y1;

  if (dx === 0 && dy === 0) {
    return Math.hypot(px - x1, py - y1);
  }

  const t = Math.max(
    0,
    Math.min(1, ((px - x1) * dx + (py - y1) * dy) / (dx * dx + dy * dy))
  );

  const projX = x1 + t * dx;
  const projY = y1 + t * dy;

  return Math.hypot(px - projX, py - projY);
}

function pickClosestConnectorSegment(connector, point, maxDistance = 12) {
  const segments = connector.geometry?.segmentsMeta || [];
  if (!segments.length) return null;

  let best = null;
  let bestDist = Infinity;

  for (const seg of segments) {
    const d = distToSegment(
      point.x,
      point.y,
      seg.x1,
      seg.y1,
      seg.x2,
      seg.y2
    );

    if (d < bestDist) {
      bestDist = d;
      best = seg;
    }
  }

  if (!best || bestDist > maxDistance) {
    return null;
  }

  return best;
}
```

## 2. `handleConnectorContextMenu`

### заміни на таке

```js id="ctx-seg-02"
handleConnectorContextMenu(e, connector) {
  e.evt.preventDefault();

  const store = useCanvasStore.getState();
  if (!connector || connector.kind !== "connector") return;

  const stage = e.target.getStage();
  const pointer = stage?.getPointerPosition();
  if (!pointer) return;

  const clickedSegment = pickClosestConnectorSegment(connector, pointer, 12);

  store.setContextMenu?.({
    type: "connector",
    edgeId: connector.id,
    segmentId: clickedSegment?.id || null,
    x: pointer.x,
    y: pointer.y,
  });
}
```

## 3. якщо хочеш заборонити add point на дуже коротких сегментах

Ось мінімальна перевірка:

```js id="ctx-seg-03"
function segmentLength(seg) {
  return Math.abs(seg.x2 - seg.x1) + Math.abs(seg.y2 - seg.y1);
}
```

І тоді в `handleConnectorContextMenu`:

```js id="ctx-seg-04"
const clickedSegment = pickClosestConnectorSegment(connector, pointer, 12);

const allowAddPathPoint =
  clickedSegment && segmentLength(clickedSegment) >= 24;
```

і в payload:

```js id="ctx-seg-05"
store.setContextMenu?.({
  type: "connector",
  edgeId: connector.id,
  segmentId: allowAddPathPoint ? clickedSegment.id : null,
  x: pointer.x,
  y: pointer.y,
});
```

## 5. що це дає

Після цього:

* контекстне меню чіпляється до **реального сегмента** з `segmentsMeta`
* в payload іде **`segmentId`**
* `addConnectorPathHandleFromContextMenu()` уже зможе працювати напряму
* можна додавати point і на **newly created inner segment**

---

## 6. якщо клік не завжди влучає

Підніми `maxDistance`:

* `12` → нормальний старт
* `16` → легше влучати
* `20` → якщо товста лінія або масштаб
